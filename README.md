# Best web IDE like vim's dot file

> This steps are for unix system if you want to use it in another environment use each plugin's site

## how to install?
  - clone the repo
  - cd vimdot
  - cp ~/.vimrc ~/.vimrc_bk
  - cp vimrc ~/.vimrc
  - mv ~/.vim ~/.vim_bk
  - cp vim ~/.vim
  - cd .. && rm -rf vimdot
  - Ready to use
